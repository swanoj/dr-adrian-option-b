# assets-master

Consolidated Wix export and website asset package for `dr-adrian-option-b`.

## Folder map

- `logos/brand`: Primary project logo files used in the rebuild
- `logos/wix-logos`: Supplied Wix logo SVG sets
- `logos/associations`: Association and institution logos (AMA, UNICEF, RMIT, etc.)
- `fonts`: Exported web font files
- `hero`: Hero-specific image and lockup assets
- `sections`: Reusable section artwork and panel assets
- `photos`: Source photo assets
- `health-logos`: Health fund and insurer logos
- `raw-exports/wix-export-20260528`: Raw export bundle
- `raw-exports/wix-export-20260528-clean`: Cleaned export bundle
- `raw-exports/drad-c2-v25-2-temp`: Temporary export/work folder
- `manifests/assets-master-file-list.txt`: Full file index

## Notes

- This directory was created to keep all migration assets in one place for handoff and ongoing development.
- The live site implementation assets are still served from `public/` in this repo.
